# TextFlow 🌐
**Advanced Text Translation & String Generation Platform**

Transform your text across 20+ languages with AI-powered translation and generate secure random strings—all in one powerful, privacy-first application.

## 🚀 What is TextFlow?

TextFlow revolutionizes text processing by combining intelligent translation with advanced string generation in a seamless, user-friendly platform:

✅ **AI-Powered Translations** - Instant, accurate translations across 20+ languages  
🔐 **Secure String Generation** - Cryptographically secure random strings with custom parameters  
🎨 **Modern UI/UX** - Clean, responsive design with dark mode support  
⚡ **Real-time Processing** - Lightning-fast translations and generation  
📱 **Mobile-First** - Optimized for all devices and screen sizes  
🔒 **Privacy-Focused** - No data storage, client-side processing when possible

## 🗺️ User Flow

The following illustrates the TextFlow user journey:

\`\`\`
1️⃣ User enters text or selects generation mode
2️⃣ Chooses target language or string parameters  
3️⃣ Clicks translate/generate or uses keyboard shortcut
4️⃣ Results appear instantly with copy functionality
5️⃣ History is saved locally for future reference
6️⃣ Export or share results as needed
\`\`\`

**Step-by-step Translation:**
1. **Input** → Enter English text in the translator
2. **Select** → Choose target language from 20+ options  
3. **Translate** → Get instant AI-powered translation
4. **Copy** → One-click clipboard integration
5. **History** → Access previous translations anytime

**Step-by-step String Generation:**
1. **Configure** → Set length (1-100) and character types
2. **Generate** → Create cryptographically secure strings
3. **Customize** → Include/exclude uppercase, numbers, symbols
4. **Save** → Automatic history with local storage
5. **Export** → Bulk copy or individual string selection

## 📦 Quick Start

\`\`\`bash
# Clone the repository
git clone https://github.com/yourusername/textflow.git
cd textflow

# Install dependencies
npm install

# Set up environment (optional for demo mode)
echo "NEXT_PUBLIC_RAPIDAPI_KEY=your_key_here" > .env.local

# Start development server
npm run dev

# Open http://localhost:3000
\`\`\`

## 🔧 Environment Setup

| Variable | Description | Required |
|----------|-------------|----------|
| `NEXT_PUBLIC_RAPIDAPI_KEY` | RapidAPI key for Google Translate | Optional* |

*Demo mode provides full functionality without API key

### 🔑 Getting Your API Key

1. **Sign up** at [RapidAPI.com](https://rapidapi.com)
2. **Subscribe** to Google Translate API (free tier available)
3. **Copy** your API key from the dashboard
4. **Add** to your environment variables
